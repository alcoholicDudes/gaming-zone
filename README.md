#  लघु-link
a URL shortener that doesn't suck

## Installation

Use yarn package manager [yarn](https://classic.yarnpkg.com/en/docs/install/#debian-stable) to install Java-script dependencies

```sh
yarn
```

## Running project

```sh
yarn start
```
